Herbal Adulteration Detection - Guide

Step 1: Setup Hardware

  • Connect ESP32
  • Connect RGB sensor, MQ135, DHT11

Step 2: Upload Code

  • Open Arduino IDE
  • Select board as ESP32 Dev Module
  • Select Port as usbserial-0001
  • Upload code
  • Change bruad rate to 115200 in Serial Monitor

Step 3: Connect WiFi

  • Enter SSID and Password
  • Enter channelID and WriteAPI key from the ThingSpeak cloud

Step 4: ThingSpeak Setup

  • Create account and login
  • Create channel and add channel name as herb
  • Add 8 fields such as:
  • Red Value
  • Green Value
  • Blue Value
  • Gas Value
  • Temperature
  • Humidity
  • G/R ratio
  • Result
  • Make the channel as public view

Step 5: View Dashboard

  • Open dashboard HTML file
  • Copy the Read API key from ThingSpeak cloud and paste it in the HTMLcode
  • See live graphs and result
